# Gym Membership and Class Management System (GMCS)

The **Gym Membership and Class Management System (GMCS)** is a desktop application developed using **Java Swing** and a **database backend** to modernize gym operations. It provides a centralized platform for managing memberships, class scheduling, trainer appointments, attendance, payments, and member fitness progress.

The system replaces manual record keeping with an organized digital solution, allowing members to register, renew memberships, book classes, monitor their progress, and view attendance records. Administrators and trainers can efficiently manage member information, schedules, payments, and gym activities, reducing administrative workload while improving accuracy and overall user experience.

Designed for small to medium-sized fitness centers, GMCS offers a secure, scalable, and user-friendly solution that enhances both operational efficiency and member engagement.

---

## Features

- Member registration and account management
- Membership renewal and payment management
- Class scheduling and booking
- Personal trainer appointment requests
- Attendance tracking
- Fitness progress monitoring
- Administrator dashboard
- Trainer dashboard
- Member dashboard
- Member and trainer profile management
- Database-driven data management
- User-friendly Java Swing interface

---

## Tech Stack

- **Java**
- **Java Swing**
- **JDBC**
- **MySQL**
- **XAMPP**
